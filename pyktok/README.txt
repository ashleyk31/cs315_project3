Steps to run PykTok:
1. cd into the data_collection folder
2. python -m venv .project3  (You'll create a new virtual environment to collect data with pyktok)
3. source .project3/bin/activate (activate it)
4. pip install -r requirements.txt (will install all packaged you need)
6. python pyktok-collect.py tiktok_history_sample.json results.csv
